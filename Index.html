Bienvenidos a nuestra web

Los módulos sensométricos y psicométricos de JT- Tecnología Aplicada conforman un conjunto de intrumentos que permiten evaluar y determinar el grado de aptitud de cada conductor, revisando sus funciones psicotécnicas y sensomotrices en un tiempo acotado.

Exámen Teórico Nuevo

El sistema de examen teórico de JT es una plataforma 100 % autoadministrable y sencilla, pensada para agilizar la toma de exámenes para obtener la licencia de conducir. Cada examen es irrepetible ya que se genera de manera aleatoria. Previo a la instancia teórica definitiva, el usuario podrá realizar simulaciones virtuales mediante la versión web.

COVID 19

Al utilizar el Gabinete Psicosensométrico nos cuidamos entre todos ya que se mantiene la distancia necesaria entre examinador y examinado, es fácil y rápida de sanitizar y el tiempo de evaluación es adaptable a las recomendaciones dadas por el área de salud.
